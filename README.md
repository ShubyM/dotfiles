# Dotfiles

My tmux, neovim, and ghostty configs.

## Contents

- `.tmux.conf` - Tmux configuration
- `.config/nvim/` - Neovim configuration
- `.config/ghostty/` - Ghostty terminal configuration

## Installation

```bash
git clone https://github.com/ShubyM/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```
